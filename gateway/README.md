# Gateway Module

API Gateway para el monolito modular, construido con Spring Boot WebFlux.

## Funcionalidades
- Enrutamiento de requests
- Filtros de autenticación
- Rate limiting
- Load balancing

## Tecnologías
- Spring Boot WebFlux
- Spring Cloud Gateway
- Reactor

## Uso en el monolito
Este módulo se integra como subtree en el proyecto principal.
